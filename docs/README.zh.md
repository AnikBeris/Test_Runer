## 📊 支持的架构和设备

<details> 
    <summary>⚙️ 展开内容</summary>

我们的平台提供与广泛架构和设备的兼容性，可适配各种计算环境。以下是我们支持的主要架构：

- **amd64:** 此常见架构为个人电脑和服务器的标准配置，并支持大多数现代操作系统。

- **x86 / i386:** 常用于桌面电脑和笔记本。支持多种操作系统和应用程序，包括 `Windows、macOS、Linux` 等。

- **armv8 / arm64 / aarch64:** 面向现代手机和嵌入式设备，例如智能手机和平板电脑。设备包括 `Raspberry Pi 4、Raspberry Pi 3、Raspberry Pi Zero 2/Zero 2 W`、`Orange Pi 3 LTS` 等。

- **armv7 / arm / arm32:** 面向更早期的手机及嵌入式设备设备，如 `Orange Pi Zero LTS、Orange Pi PC Plus、Raspberry Pi 2` 等仍被广泛使用。

- **armv6 / arm / arm32:** 主要用于非常老旧的嵌入设备，例如 `Raspberry Pi 1、Raspberry Pi Zero/Zero W`。

- **armv5 / arm / arm32:** 更早期的架构，主要用于早期嵌入式系统。如早期版本的 `Raspberry Pi` 和旧款智能手机。

- **s390x:** 这类架构通常用于 IBM 主机，为企业负载提供高性能和可靠性支持。

</details>

SSL证书 Cloudflare
</h1>

<div align="center">
  <img src="./media/Tutorial/Article_1/cloudflare.jpg" alt=" Cloud flare " width="50%">
</div>

管理脚本内置通过 `Cloudflare` 请求 `SSL证书` 的功能。要使用此脚本获取证书，您需要：


<details> 
    <summary> ⚙️ 展开内容 </summary>

- 注册 `Cloudflare` 的电子邮箱
- Global API Key Cloudflare
- 域名需通过 `Cloudflare` 定向到当前服务器（在 DNS 中指定）

## 如何获取 Global API Key Cloudflare:

1. 在终端执行命令 `x-ui`，然后选择 `Cloudflare SSL Certificate`。

2. 点击链接：[Cloudflare API Tokens](https://dash.cloudflare.com/profile/api-tokens)

3. 点击 `View Global API Key`（见下图）：

<div align="center">
  <img src="./media/Tutorial/Article_1/APIKey1.png" alt=" API Key 1 " width="70%">
</div>

4. 您可能需要重新验证身份。验证后显示密钥（见下图）：

<div align="center">
  <img src="./media/Tutorial/Article_1/APIKey2.png" alt=" API Key 2 " width="70%">
</div>


使用时输入您的域名、`email` 和 `API KEY`。示例如下：

安装 3X-UI
</h1>


# 1. 3X快速安装

<details> 
    <summary> ⚙️ 展开内容 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh

```

</details> 

# 2. 3X分步安装


<details> 
    <summary> ⚙️ 展开内容 </summary>

# 📊 安装必要的包

1. 系统更新
在安装开始之前，确保您的系统是最新的。执行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装必要的包

确保您的服务器安装了所需的包：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装 3X-UI 面板

将在服务器上安装面板，请通过以下脚本命令启动安装：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装过程中脚本将提问：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

回答 `y` 自定义端口，或 `n` 让脚本随机选择端口
请勿使用默认端口 `22、80、8080`。建议选择其他端口，例如 `8181`


```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

完成以上步骤后，面板安装完成，可以通过以下路径浏览器连接到面板

```sh
http://服务器IP地址:端口号/WebBasePath(例: http://192.168.0.10:40608/vpkPI6ex9ajesDX)
```

在浏览器输入地址后，您将进入登录面板，输入脚本生成并提示的用户名及密码即可登录。
