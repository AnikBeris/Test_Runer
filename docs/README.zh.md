## 📊 支持的架构与设备

<details> 
    <summary>⚙️ 展开说明</summary>

我们的平台支持广泛的架构和设备，适用于各种计算环境。以下为主要支持的架构列表：

- **amd64:** 此主流架构是个人电脑和服务器的标准，适用于大多数现代操作系统。

- **x86 / i386:** 广泛用于台式电脑和笔记本电脑。此架构支持许多操作系统和应用，包括 `Windows, macOS 和 Linux`。

- **armv8 / arm64 / aarch64:** 针对现代移动设备和嵌入式设备，诸如智能手机与平板电脑。示例设备：`Raspberry Pi 4, Raspberry Pi 3, Raspberry Pi Zero 2/Zero 2 W`, `Orange Pi 3 LTS`等。

- **armv7 / arm / arm32:** 用于较老的移动及嵌入式设备。依然广泛用于设备如`Orange Pi Zero LTS、Orange Pi PC Plus、Raspberry Pi 2`等。

- **armv6 / arm / arm32:** 针对非常老旧的嵌入式设备，尽管较少见，仍用于例如`Raspberry Pi 1, Raspberry Pi Zero/Zero W`。

- **armv5 / arm / arm32:** 一种较老的架构，主要用于早期的嵌入式系统。现较少见，但可能在一些早期`Raspberry Pi`及旧款智能手机中使用。

- **s390x:** 此架构通常用于`IBM`大型机，提供企业级任务的高性能及可靠性。

</details>

## 📊 功能或特点

<details> 
    <summary>⚙️ 展开说明</summary>

- 系统状态监控
- 搜索所有入站连接及客户端
- 主题: `暗色/亮色`
- 支持多用户及多协议
- 支持协议: `VMESS, VLESS, Trojan, Shadowsocks, Dokodemo-door, Socks, HTTP, WireGuard`
- 支持原生 XTLS 协议: `RPRX-Direct, Vision, REALITY`
- 数据流统计、流量限制及时间到期限制
- 可自定义 `Xray` 配置模板
- 通过 `HTTPS (自定义域名 + SSL 证书)`访问面板支持
- 支持一键申请`SSL证书`及其自动续期
- 更多高级配置参数请访问面板
- 修正的 `API 路由` (通过 `API` 添加用户配置)
- 支持基于不同参数的配置修改
- 支持通过面板导入/导出数据库

</details>

管理脚本包括通过 `Cloudflare` 申请 `SSL 证书` 的内置功能。要使用此脚本获取证书，您需要：


<details> 
    <summary> ⚙️ 展开说明 </summary>

- 已注册在 `Cloudflare` 的电子邮件
- Cloudflare 的 Global API Key
- 通过 `Cloudflare` 将域名在 DNS 中指向当前服务器



## 如何获得 Cloudflare 的 Global API Key:

1. 在终端执行命令 `x-ui`，然后选择 `Cloudflare SSL Certificate`。

2. 访问链接: [Cloudflare API Tokens](https://dash.cloudflare.com/profile/api-tokens)

3. 点击 `View Global API Key` (见下图):

</div>

4. 可能需要重新认证。完成后将显示密钥 (见下图):

<div align="center">

<h1 align="center"> 
3X-UI 安装
</h1>


# 1. 一键快速安装 3X

<details> 
    <summary> ⚙️ 展开说明 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh

```

</details> 

# 2. 分步骤安装 3X


<details> 
    <summary> ⚙️ 展开说明 </summary>

# 📊 安装所需软件包

1. 更新系统
在开始安装之前，确保您的系统为最新版本。执行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装所需软件包

确保您的服务器中已安装必需的软件包：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装 3X-UI 面板

在服务器上执行以下命令安装面板：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装过程中脚本会询问：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

回答 `y` 自定义设置端口，回答 `n` 则脚本会随机生成端口。
不要使用默认端口 `22, 80, 8080`。建议使用任意其它端口，例如 `8181`。


```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

完成上述步骤后，面板安装完成。可以通过以下地址在浏览器中访问面板：

```sh
http://您的服务器IP地址:端口/WebBasePath (示例: http://192.168.0.10:40608/vpkPI6ex9ajesDX)
```

在浏览器中输入地址后，您将进入面板登录界面，输入之前脚本提供的用户名和密码。
