## 📊 支持的架构和设备

<details> 
    <summary>⚙️ 展开说明</summary>

我们的平台兼容广泛的架构和设备，能适应不同的运算环境。以下是我们支持的主要架构：

- **amd64:** 这种广泛使用的架构是个人电脑和服务器的标准，能够无缝运行大部分现代操作系统。

- **x86 / i386:** 常用于台式电脑和笔记本电脑。这种架构在多个操作系统及应用程序中广泛支持，包括 `Windows, macOS 和 Linux`。

- **armv8 / arm64 / aarch64:** 专为现代移动和嵌入式设备设计，如智能手机和平板电脑。设备示例：`Raspberry Pi 4, Raspberry Pi 3, Raspberry Pi Zero 2/Zero 2 W`, `Orange Pi 3 LTS` 等。

- **armv7 / arm / arm32:** 是较老的移动设备和嵌入式设备的架构。目前仍广泛用于设备，比如 `Orange Pi Zero LTS, Orange Pi PC Plus, Raspberry Pi 2` 等。

- **armv6 / arm / arm32:** 面向非常老旧的嵌入式设备。尽管相对较少见，仍用于像 `Raspberry Pi 1, Raspberry Pi Zero/Zero W` 这样的设备。

- **armv5 / arm / arm32:** 更古老的架构，通常与最早期的嵌入式系统相关。如今已经少见，但可能应用于早期 `Raspberry Pi` 设备和一些旧智能手机中。

- **s390x:** 该架构主要用于 `IBM` 大型机，为企业级工作负载提供高性能和高可靠性。

</details>

## 📊 功能 || 特点

<details> 
    <summary>⚙️ 展开说明</summary>

- 系统状态监控
- 所有入站连接与客户端的搜索功能
- 主题支持：`暗黑/明亮`

- 更多高级配置选项可在面板中查看
- 修复的 `API 路由` （用户设置通过 `API` 创建）
- 支持根据面板中可用的不同参数更改配置
- 支持通过面板导入/导出数据库

</details>

<details>

- 注册在 `Cloudflare` 的电子邮件地址
- Global API Key Cloudflare
- 域名必须通过 `Cloudflare` 指向（在DNS中记录）当前服务器

安装 3X-UI
</h1>


# 1. 快速安装 3X

<details> 
    <summary> ⚙️ 展开说明 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh

```

</details> 

# 2. 分步安装 3X


<details> 
    <summary> ⚙️ 展开说明 </summary>

# 📊 安装必要的包

1. 系统更新
在安装之前，请确保您的系统是最新的。运行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装必要的软件包

确保您的服务器上已安装必要的软件包：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 3X-UI 面板安装

要在服务器上安装面板，请运行以下脚本命令：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装过程脚本会提示您：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

如果您想手动设置端口，回答 `y`；否则回答 `n`，脚本会自动分配一个随机端口。
请勿使用标准端口（例如 `22, 80, 8080`）。推荐设置其他端口，比如 `8181`。


```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

完成这些步骤后，面板安装完成，您可以通过浏览器访问面板，路径为：

```sh
http://服务器的IP地址:端口/WebBasePath (例如: http://192.168.0.10:40608/vpkPI6ex9ajesDX)
```

输入地址后，您会进入面板登录页面，在此输入脚本之前提供的用户名和密码。
