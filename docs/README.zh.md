## 📊 支持的架构和设备

<details> 
    <summary>⚙️ 展开描述</summary>

我们的平台兼容多种架构和设备，以便在各种计算环境中提供灵活性。以下列出我们支持的主要架构：

- **amd64:** 一种常见架构，主要应用于个人计算机和服务器，支持大部分现代操作系统。

- **x86 / i386:** 广泛用于台式机和笔记本，兼容多种操作系统和应用程序，包括 `Windows, macOS 和 Linux`。

- **armv8 / arm64 / aarch64:** 专为现代移动设备和嵌入式设备设计，如智能手机和平板电脑，支持设备包括 `Raspberry Pi 4, Raspberry Pi 3, Raspberry Pi Zero 2/Zero 2 W`，`Orange Pi 3 LTS` 等。

- **armv7 / arm / arm32:** 服务于较旧的移动和嵌入式设备，应用于如 `Orange Pi Zero LTS, Orange Pi PC Plus, Raspberry Pi 2` 等设备。

- **armv6 / arm / arm32:** 适用于非常旧的嵌入式设备，例如 `Raspberry Pi 1, Raspberry Pi Zero/Zero W`。

- **armv5 / arm / arm32:** 较早的架构，主要与早期嵌入式系统相关。尽管较少使用，但仍可能用于较老设备如早期版本的 `Raspberry Pi` 和一些旧型号的智能手机。

- **s390x:** 常用于 `IBM` 主机，提供企业工作负载所需要的高性能和高可靠性。

</details>

安装 3X-UI

# 1. 快速安装 3X

<details> 
    <summary> ⚙️ 展开描述 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh

```

</details>

<details> 
    <summary> ⚙️ 展开描述 </summary>

# 📊 安装必要的包

1. 更新系统
在安装之前确保系统是最新的，执行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装必要的包

确保您的服务器已安装相关依赖：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装 3X-UI 面板

要安装面板到服务器上，执行以下命令：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装过程中，脚本将提出如下问题：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

选择 `y` 可自定义端口，选择 `n` 则脚本自动分配端口。
请勿使用默认端口 `22, 80, 8080`。建议选择其他端口，例如 `8181`。

```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

在完成这些步骤后，面板安装完成，您可以通过浏览器访问面板：

```sh
http://服务器IP地址:端口/WebBasePath（例：http://192.168.0.10:40608/vpkPI6ex9ajesDX）
```

输入浏览器地址后，系统将跳转至面板登录页面，您需要输入脚本安装期间给出的用户名和密码。
