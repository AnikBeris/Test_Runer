## 📊 支持的架构与设备

<details> 
    <summary>⚙️ 展开详细信息</summary>

我们的平台支持广泛的架构和设备，提供多种计算环境的灵活性。以下列举了主要支持的架构：

- **amd64:** 这种广泛使用的架构是个人电脑和服务器的标准，支持大多数现代操作系统的无缝运行。

- **x86 / i386:** 广泛用于台式机和笔记本电脑。此架构得到许多操作系统和应用程序的广泛支持，包括但不限于 `Windows, macOS 和 Linux`。

- **armv8 / arm64 / aarch64:** 面向现代移动和嵌入式设备（如智能手机和平板电脑）。设备示例：`Raspberry Pi 4, Raspberry Pi 3, Raspberry Pi Zero 2/Zero 2 W`, `Orange Pi 3 LTS` 等。

- **armv7 / arm / arm32:** 用于老一代的移动和嵌入式设备，仍广泛应用于如 `Orange Pi Zero LTS, Orange Pi PC Plus, Raspberry Pi 2` 等设备。

- **armv6 / arm / arm32:** 面向非常老旧的嵌入式设备。尽管应用较少，仍然在如 `树莓派 1, 树莓派 零/零 W` 中使用。

- **armv5 / arm / arm32:** 较早的架构，主要用于早期嵌入式系统。如今较为少见，但可能存在于一些过时设备中，如早期版的 `树莓派` 和一些旧型号智能手机。

- **s390x:** 通常用于 `IBM` 的大型主机，为企业工作负载提供高性能和高度可靠。

</details>

安装3X-UI
</h1>


# 1. 快速安装3X

<details> 
    <summary> ⚙️ 展开详细信息 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh

```

</details> 

# 2. 分步安装3X


<details> 
    <summary> ⚙️ 展开详细信息 </summary>

# 📊 安装必要包

1. 更新系统
在开始安装之前，确保系统已更新。运行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装必要包

确保您的服务器安装了相关必要包：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装控制面板3X-UI

在服务器上运行以下命令以安装控制面板：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装过程中脚本将询问以下问题：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

选择`y`手动设置端口，或者选择`n`让脚本自行分配端口。
请勿设置默认端口`22, 80, 8080`，推荐选择其他端口，如`8181`。


```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

完成这些步骤后，安装控制面板完成，可以通过以下路径通过浏览器连接到控制面板：

```sh
http://您的服务器IP:端口/WebBasePath(示例: http://192.168.0.10:40608/vpkPI6ex9ajesDX)
```

在浏览器中输入地址后，您将进入登录页面。输入脚本提供的用户名和密码登录。
