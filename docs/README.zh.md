## 📊 支持的架构与设备

<details> 
    <summary>⚙️ 展开描述</summary>

我们的平台与众多架构和设备兼容，提供在不同计算环境下的灵活性。以下列出了我们支持的主要架构：

- **amd64:** 常见架构，是个人电脑和服务器的标准，确保现代操作系统可无缝运行。

- **x86 / i386:** 广泛用于台式电脑和笔记本。多种操作系统与应用程序支持此架构，例如`Windows, macOS 和 Linux`。

- **armv8 / arm64 / aarch64:** 专为现代移动和嵌入式设备设计，如智能手机和平板电脑。设备示例：`Raspberry Pi 4, Raspberry Pi 3, Raspberry Pi Zero 2/Zero 2 W`, `Orange Pi 3 LTS`等。

- **armv7 / arm / arm32:** 用于较旧的移动和嵌入式设备。仍广泛应用于诸如`Orange Pi Zero LTS, Orange Pi PC Plus, Raspberry Pi 2`等设备。

- **armv6 / arm / arm32:** 针对非常旧的嵌入式设备。尽管较少见，仍用于如`Raspberry Pi 1, Raspberry Pi Zero/Zero W`等设备。

- **armv5 / arm / arm32:** 较早架构，多与早期嵌入式系统相关。现已较少见，但可能用于如早期版本`Raspberry Pi`的一些遗留设备。

- **s390x:** 此架构通常应用于`IBM`的主机计算环境，能够提供高性能及可靠性以满足企业工作负载需求。

</details>

安装3X-UI
</h1>


# 1. 快速安装3X

<details> 
    <summary> ⚙️ 展开描述 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh

```

</details> 

# 2. 分步安装3X


<details> 
    <summary> ⚙️ 展开描述 </summary>

# 📊 安装必要的包

1. 更新操作系统
在开始安装之前，请确保您的系统是最新版本。执行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装必需的软件包

确保您的服务器安装了以下必要软件包：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装3X-UI面板

想要在服务器上安装面板，请运行以下命令：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装过程中脚本会询问：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

如果想要自己设置端口请回答`y`，否则选择`n`，脚本将随机分配端口。
避免使用默认端口`22, 80, 8080`，推荐选择其他端口，如`8181`


```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

完成以上步骤后，面板安装完成，可以通过浏览器访问如下路径：

```sh
http://您的服务器IP地址:端口/WebBasePath（例如：http://192.168.0.10:40608/vpkPI6ex9ajesDX）
```

输入地址后将进入面板登录界面，输入用户名和密码即可登录，相关信息已在安装脚本中显示。
