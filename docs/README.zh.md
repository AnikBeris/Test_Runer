## 📊 支持的架构和设备

<details> 
    <summary>⚙️ 展开内容</summary>

我们的平台提供对各种架构和设备的兼容性，确保在不同计算环境下的灵活性。以下列出了主要支持的架构：

- **amd64:** 这种广泛使用的架构是个人计算机和服务器的标准，提供对大多数现代操作系统的无缝支持。

- **x86 / i386:** 广泛用于台式计算机和笔记本电脑。此架构得到了许多操作系统和应用程序的广泛支持，包括`Windows, macOS 和 Linux`等。

- **armv8 / arm64 / aarch64:** 为现代移动和嵌入式设备（如智能手机和平板电脑）设计的架构。示例设备包括：`Raspberry Pi 4, Raspberry Pi 3, Raspberry Pi Zero 2/Zero 2 W`, `Orange Pi 3 LTS`等。

- **armv7 / arm / arm32:** 用于较旧移动和嵌入式设备的架构，仍然广泛应用于`Orange Pi Zero LTS, Orange Pi PC Plus, Raspberry Pi 2`等设备。

- **armv6 / arm / arm32:** 针对古老嵌入式设备的架构，虽较少见，但仍被使用，例如`Raspberry Pi 1, Raspberry Pi Zero/Zero W`。

- **armv5 / arm / arm32:** 较旧的架构，主要在早期嵌入式系统中使用。现较少见，但可能被应用于过时设备，如早期的`Raspberry Pi`或一些旧智能手机。

- **s390x:** 通常用于`IBM`大型主机的架构，为企业工作负载提供高性能和可靠性。

</details>

## 📊 功能 || 特点

<details> 
    <summary>⚙️ 展开内容</summary>

- 系统状态监控
- 查询所有传入连接及客户端
- 主题支持：`深色/亮色`
- 支持多用户及多协议
- 支持协议：`VMESS, VLESS, Trojan, Shadowsocks, Dokodemo-door, Socks, HTTP, WireGuard`
- 支持原生 XTLS 协议：`RPRX-Direct, Vision, REALITY`
- 流量统计、流量限制、到期时间限制
- 可自定义 Xray 配置模板
- 支持面板通过`HTTPS（自定义域名 + SSL 证书）`访问
- 支持一键申请`SSL 证书`及自动续期
- 用于更高级的配置选项详见面板
- 修复的`API 路由`（通过 `API` 创建用户配置）
- 支持针对配置面板中的多种参数修改配置
- 支持通过面板导出/导入数据库

</details>

<details>

<h1 align="center"> 
安装 3X-UI
</h1>


# 1. 快速安装 3X

<details> 
    <summary> ⚙️ 展开内容 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh

```

</details> 

# 2. 分步安装 3X


<details> 
    <summary> ⚙️ 展开内容 </summary>

# 📊 安装必要的软件包

1. 更新系统
在安装前，确保系统已更新。执行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装必要的软件包

确保您的服务器已安装以下必要包：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装 3X-UI 面板

在服务器上运行以下安装脚本命令：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装过程中脚本会询问：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

选择`y`手动设置端口，或选择`n`由脚本自动设置端口。
不要使用默认端口`22, 80, 8080`。建议改用其他端口，例如`8181`。


```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

完成上述步骤后，面板安装完成。可以通过浏览器访问以下路径的面板：

```sh
http://您的服务器IP地址:端口/WebBasePath(例如: http://192.168.0.10:40608/vpkPI6ex9ajesDX)
```

输入面板地址后，您将看到登录界面，请输入脚本提供的用户名和密码。
