## 📊 支持的架构和设备

<details> 
    <summary>⚙️ 展开说明</summary>

我们的平台支持各种架构和设备，能够在不同计算环境中提供灵活性。以下是我们支持的主要架构：

- **amd64:** 最常见的架构，个人电脑和服务器的标准架构，支持大多数现代操作系统。

- **x86 / i386:** 广泛应用于桌面电脑和笔记本电脑，支持多种操作系统和应用程序，例如 `Windows, macOS 和 Linux`。

- **armv8 / arm64 / aarch64:** 适用于现代的移动和嵌入式设备，例如智能手机和平板电脑。支持设备有：`Raspberry Pi 4, Raspberry Pi 3, Raspberry Pi Zero 2/Zero 2 W`, `Orange Pi 3 LTS`等。

- **armv7 / arm / arm32:** 适用于较老的移动和嵌入式设备，仍然广泛用于例如 `Orange Pi Zero LTS, Orange Pi PC Plus, Raspberry Pi 2`等。

- **armv6 / arm / arm32:** 用于非常旧的嵌入式设备，虽然不常见，但仍适用于例如 `Raspberry Pi 1, Raspberry Pi Zero/Zero W`等。

- **armv5 / arm / arm32:** 更老的架构，主要与早期嵌入式系统有关，今天已经比较少见，但在过时的设备中例如早期版本的 `Raspberry Pi`和一些旧的智能手机中可能见到。

- **s390x:** 通常用于 `IBM`的大型机，提供高性能和可靠性，用于企业级负载。

</details>

## 📊 功能 || 特点

<details> 
    <summary>⚙️ 展开说明</summary>

- 系统状态监测
- 搜索所有入站连接和客户端
- 主题：`深色 / 浅色`
- 支持多用户及多协议
- 支持协议：`VMESS, VLESS, Trojan, Shadowsocks, Dokodemo-door, Socks, HTTP, WireGuard`
- 本地支持XTLS协议：`RPRX-Direct, Vision, REALITY`
- 流量统计，流量限制，过期时间限制
- 可定制`Xray`配置模板
- 支持通过 `HTTPS（自定义域名 + SSL证书）` 访问控制面板
- 支持一键申请`SSL证书`及其自动续期功能
- 更多高级配置选项请查看控制面板
- 修复的 `API路由`（用户配置通过 `API` 创建）
- 支持通过面板依据不同参数更改配置
- 支持通过面板导入/导出数据库

</details>

# 1. 快速安装3X

<details> 
    <summary> ⚙️ 展开说明 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh

```

</details> 

# 2. 分步安装3X


<details> 
    <summary> ⚙️ 展开说明 </summary>

# 📊 安装所需软件包

1. 系统更新
在开始安装之前，请确保您的系统已更新。运行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装所需的软件包

确保您的服务器上已安装所需的软件包：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装3X-UI面板

安装面板到服务器时运行以下脚本命令：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装过程中脚本会询问：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

选择`y`如果您希望手动设置端口，或者选择`n`此时脚本会自动设置端口。
请勿设置默认端口 `22, 80, 8080`。建议选择其他端口，例如 `8181`


```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

完成以上步骤后，面板安装将结束，您可以通过以下地址使用浏览器访问面板：

```sh
http://您的服务器IP地址:端口/WebBasePath(例如: http://192.168.0.10:40608/vpkPI6ex9ajesDX)
```

输入地址后会跳转到面板的登录窗口，您需要输入脚本之前提示的用户名和密码。
