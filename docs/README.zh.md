## 📊 支持的架构和设备

<details> 
    <summary>⚙️ 展开描述</summary>

我们的平台支持各种架构和设备，能够在不同的计算环境中提供灵活性。以下是我们支持的主要架构：

- **amd64:** 广泛的架构标准，适用于个人电脑及服务器，可无缝运行大多数现代操作系统。

- **x86 / i386:** 常用于台式电脑和笔记本电脑。此架构拥有多个操作系统及应用的广泛支持，如 `Windows, macOS 和 Linux`。

- **armv8 / arm64 / aarch64:** 专为现代移动及嵌入式设备设计，例如智能手机和平板电脑。设备示例：`Raspberry Pi 4, Raspberry Pi 3, Raspberry Pi Zero 2/Zero 2 W`, `Orange Pi 3 LTS` 等。

- **armv7 / arm / arm32:** 适用于较老的移动和嵌入式设备。仍广泛用于如 `Orange Pi Zero LTS, Orange Pi PC Plus, Raspberry Pi 2` 等设备。

- **armv6 / arm / arm32:** 针对非常旧的嵌入式设备。虽然不常见，但仍应用于如 `Raspberry Pi 1, Raspberry Pi Zero/Zero W` 等设备。

- **armv5 / arm / arm32:** 较老的架构，主要涉及早期的嵌入式系统。如今极为罕见，但仍可用于旧设备，如早期版本的 `Raspberry Pi` 和一些旧款智能手机。

- **s390x:** 该架构通常用于 `IBM` 主机，为企业级工作负载提供高性能和可靠性。

</details>

## 📊 功能特色

<details> 
    <summary>⚙️ 展开描述</summary>

- 系统状态监控
- 搜索所有传入连接和客户
- 主题：`暗色 /亮色`
- 支持多用户和多协议
- 支持协议：`VMESS, VLESS, Trojan, Shadowsocks, Dokodemo-door, Socks, HTTP, WireGuard`
- 支持原生 XTLS协议：`RPRX-Direct, Vision, REALITY`
- 流量统计、流量限制、到期时间限制
- 可定制的 `Xray` 配置模板
- 支持通过 `HTTPS（自定义域名+SSL证书）` 访问控制面板
- 支持单击请求 `SSL证书` 并自动续期
- 有关更高级的配置参数，请参阅面板
- 修复的 `API路线`（用户配置通过 `API` 创建）
- 支持通过面板根据各种参数修改配置
- 支持通过面板导出/导入数据库

</details>

<h1 align="center"> 
3X-UI安装
</h1>


# 1. 快速安装 3X

<details> 
    <summary> ⚙️ 展开描述 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh
```

</details> 

# 2. 分步安装 3X


<details> 
    <summary> ⚙️ 展开描述 </summary>

# 📊 安装必要的软件包

1. 更新系统
开始之前，确保您的系统是最新的，运行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装必需的软件包

确保服务器已安装以下所需软件：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装 3X-UI 面板

在服务器上执行以下命令以安装面板：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装过程中脚本会询问您：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

回答 `y` 以自定义端口，或者回答 `n` 让脚本自动生成端口。
请勿使用 `22, 80, 8080` 等默认端口，建议选择其他端口，如 `8181`。

```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

完成以上步骤后，面板安装完成，可通过浏览器访问以下地址：

```sh
http://服务器IP地址:port/WebBasePath (例如：http://192.168.0.10:40608/vpkPI6ex9ajesDX)
```

在浏览器输入该地址后，将跳转登录页面，输入脚本生成的用户名及密码即可登录。
