## 📊 支持的架构和设备

<details> 
    <summary>⚙️ 显示详情</summary>

我们的平台支持多种架构和设备配置，能够在不同的计算环境中保持灵活性。以下是可兼容的主要架构：

- **amd64:** 这种通用架构是个人计算机和服务器的标准，并很好地支持当代的大多数操作系统。

- **x86 / i386:** 广泛用于台式机和笔记本电脑。该架构得到了多种操作系统和应用程序（包括`Windows、macOS和Linux`等）的广泛支持。

- **armv8 / arm64 / aarch64:** 为现代移动和嵌入式设备（如智能手机和平板电脑）而设计。设备示例：`Raspberry Pi 4, Raspberry Pi 3, Raspberry Pi Zero 2/Zero 2 W`，`Orange Pi 3 LTS`等。

- **armv7 / arm / arm32:** 面向较旧的移动和嵌入式设备的架构。仍然广泛用于如`Orange Pi Zero LTS, Orange Pi PC Plus, Raspberry Pi 2`等设备中。

- **armv6 / arm / arm32:** 针对非常旧的嵌入式设备。尽管较少使用，但仍用于例如`Raspberry Pi 1, Raspberry Pi Zero/Zero W`。

- **armv5 / arm / arm32:** 较旧的架构，主要应用于早期嵌入式系统。如今很少见，但可能用于老旧设备，如早期的`Raspberry Pi`及一些旧款智能手机。

- **s390x:** 此架构通常用于`IBM`主机，提供企业级工作负载的高性能和可靠性。

</details>

## 📊 功能或特性

<details> 
    <summary>⚙️ 显示详情</summary>

- 系统状态监控
- 搜索所有入站连接与客户端

- 修复 `API 路由`（用户设置可通过 `API` 创建）
- 支持通过面板按多种参数更改配置
- 支持通过面板导出/导入数据库

</details>

<h1 align="center"> 
安装 3X-UI
</h1>


# 1. 快速安装 3X

<details> 
    <summary> ⚙️ 显示详情 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh

```

</details> 

# 2. 分步安装 3X


<details> 
    <summary> ⚙️ 显示详情 </summary>

# 📊 安装必要的软件包

1. 系统更新
在开始安装之前，确保您的系统已更新。运行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装必要的软件包

确保您的服务器安装了所需的软件包：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装 3X-UI 面板

在服务器上安装面板，运行以下命令：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

在安装过程中，脚本会提问：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

回答 `y` 以自定义端口，或 `n` 让脚本自动选择端口。
不要设置为标准端口 `22, 80, 8080`，建议选择其他端口，例如 `8181`。


```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

完成这些步骤后，面板安装完成，并可以通过以下路径在浏览器中访问面板：

```sh
http://您的服务器IP地址:端口/WebBasePath（例如：http://192.168.0.10:40608/vpkPI6ex9ajesDX）
```

在浏览器中输入地址后，您将看到面板的登录界面，此处需输入脚本提供的用户名和密码。
