## 📊 支持的架构和设备

<details> 
    <summary>⚙️ 展开描述</summary>

我们的平台支持多种架构和设备，为多样的计算环境提供灵活性。以下列出了我们支持的主要架构：

- **amd64:** 这是一种常见的架构，广泛用于个人计算机和服务器上，能够很好地兼容绝大多数现代操作系统。

- **x86 / i386:** 广泛应用于台式机和笔记本电脑上。这种架构得到许多操作系统和应用软件的广泛支持，包括但不限于`Windows、macOS和Linux`。

- **armv8 / arm64 / aarch64:** 用于现代移动和嵌入式设备，例如智能手机和平板电脑。设备示例：`Raspberry Pi 4、Raspberry Pi 3、Raspberry Pi Zero 2/Zero 2 W`，`Orange Pi 3 LTS`等。

- **armv7 / arm / arm32:** 用于较旧的移动和嵌入式设备。仍然广泛应用于如`Orange Pi Zero LTS、Orange Pi PC Plus、Raspberry Pi 2`等设备。

- **armv6 / arm / arm32:** 面向非常旧的嵌入式设备。尽管使用较少，但仍然应用于例如`Raspberry Pi 1、Raspberry Pi Zero/Zero W`。

- **armv5 / arm / arm32:** 一种较旧的架构，主要与早期的嵌入式系统相关。今天已经较少见到，但仍可能在过时设备中使用，例如早期版本的`Raspberry Pi`或一些老旧的智能手机。

- **s390x:** 此架构通常用于`IBM`的大型主机中，具备企业级工作负载的高性能和高可靠性。

</details>

## 📊 功能与特点

<details> 
    <summary>⚙️ 展开描述</summary>

- 系统状态监控
- 查询所有连接和客户端
- 主题：`深色模式/浅色模式`
- 支持多用户和多协议
- 支持的协议：`VMESS, VLESS, Trojan, Shadowsocks, Dokodemo-door, Socks, HTTP, WireGuard`
- 支持本地XTLS协议：`RPRX-Direct, Vision, REALITY`
- 流量统计、流量限制、到期时间限制
- 支持用户自定义`Xray`配置模板
- 支持通过`HTTPS（自定义域名+SSL证书）`访问控制面板
- 支持SSL证书一键申请及自动续订
- 更多高级配置参数请参阅控制面板
- 修复的`API路由`（可通过`API`创建用户设置）
- 支持通过控制面板修改基于不同参数的配置
- 支持通过控制面板导入/导出数据库

</details>

# 📊 安装必要包

1. 系统更新
在开始安装之前，请确保您的系统是最新的。运行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装必要包

确保您的服务器安装了所需的包：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装3X-UI面板

在服务器上运行以下命令安装面板：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装过程中，脚本会提问：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

回答`y`表示您想自定义端口设置，或回答`n`让脚本自动选择端口。
不要使用常见的端口，例如`22, 80, 8080`。推荐选择其他端口，例如`8181`。


```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

完成这些步骤后，面板安装将完成，可以通过浏览器访问以下路径进入面板：

```sh
http://您的服务器IP地址:端口号/WebBasePath（例如：http://192.168.0.10:40608/vpkPI6ex9ajesDX）
```

在浏览器输入地址后，您将看到面板的登录页面，输入之前脚本提供的用户名和密码即可登录。
