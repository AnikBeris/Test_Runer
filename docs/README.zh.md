## 📊 支持的架构和设备

<details> 
    <summary>⚙️ 点击展开</summary>

我们的平台支持众多架构和设备，能够适应不同的计算环境。以下列出主要支持的架构：

- **amd64:** 这一常见架构是个人电脑和服务器的标准，大多数现代操作系统都支持。

- **x86 / i386:** 广泛用于桌面计算机和笔记本电脑。支持多种操作系统和应用，如`Windows, macOS 和 Linux`。

- **armv8 / arm64 / aarch64:** 适用于现代移动和嵌入式设备，如智能手机和平板电脑。设备例子包括`Raspberry Pi 4, Raspberry Pi 3, Raspberry Pi Zero 2/Zero 2 W`, `Orange Pi 3 LTS`等。

- **armv7 / arm / arm32:** 是老版移动和嵌入式设备的架构。仍广泛用于设备如`Orange Pi Zero LTS, Orange Pi PC Plus, Raspberry Pi 2`等。

- **armv6 / arm / arm32:** 专为非常老旧的嵌入式设备设计。尽管使用较少，仍应用于例如`Raspberry Pi 1, Raspberry Pi Zero/Zero W`等设备。

- **armv5 / arm / arm32:** 较为老旧的架构，以早期嵌入式系统为主，目前很少见，但可用于一些旧设备如早期`Raspberry Pi`型号和某些老旧智能手机。

- **s390x:** 通常用于`IBM`的大型机，为企业工作负载提供高性能和可靠性。

</details>

## 📊 功能 || 特点

<details> 
    <summary>⚙️ 点击展开</summary>

- 系统状态监控
- 搜索所有传入连接和客户端

- 支持通过面板修改多项配置参数
- 支持通过面板导出/导入数据库

</details>

控制脚本包括内置功能，可通过`Cloudflare`请求`SSL证书`。使用脚本获取证书需要以下内容：


<details> 
    <summary> ⚙️ 点击展开 </summary>

- `Cloudflare`注册的电子邮箱
- Global API Key Cloudflare
- 域名需通过`Cloudflare`指向当前服务器（DNS记录要求）



## 如何获取 Global API Key Cloudflare:

1. 在终端运行命令`x-ui`，然后选择`Cloudflare SSL Certificate`

3. 点击`View Global API Key`（参考下方截图）：

<h1 align="center"> 
安装 3X-UI
</h1>


# 1. 快速安装 3X

<details> 
    <summary> ⚙️ 点击展开 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh

```

</details> 

# 2. 分步安装 3X


<details> 
    <summary> ⚙️ 点击展开 </summary>

# 📊 安装必要包

1. 更新系统
开始安装之前，请确保系统已经更新。运行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装必要包

确保服务器已安装以下必要组件：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装 3X-UI 面板

在服务器运行以下脚本以安装面板：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装期间，脚本会提问：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

回答`y`以自定义面板端口，或回答`n`让脚本自行分配端口。
请勿使用默认端口`22, 80, 8080`。建议选择其他端口，例如`8181`。

```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

完成这些步骤后即可安装面板，通过以下路径用浏览器连接到面板：

```sh
http://服务器IP地址:端口号/WebBasePath(例: http://192.168.0.10:40608/vpkPI6ex9ajesDX)
```

在浏览器输入地址后，会进入面板登录页面，请输入脚本前提供的用户名和密码。
