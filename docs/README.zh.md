## 📊 支持的架构和设备

<details> 
    <summary>⚙️ 展开描述</summary>

我们的平台兼容多种架构和设备，适用于多种计算环境。以下是主要支持的架构：

- **amd64：** 常用架构，是个人电脑和服务器的标准架构，可支持大多数现代操作系统。

- **x86 / i386：** 广泛用于台式机和笔记本电脑，支持多种操作系统和应用。

- **armv8 / arm64 / aarch64：** 针对现代移动和嵌入式设备，如智能手机和平板电脑的架构。

- **armv7 / arm / arm32：** 适用于较旧的移动和嵌入式设备，如`Orange Pi Zero LTS, Raspberry Pi 2`等。

- **armv6 / arm / arm32：** 针对更老的嵌入式设备，如`Raspberry Pi 1, Raspberry Pi Zero/Zero W`。

- **armv5 / arm / arm32：** 较古老的架构，主要用于早期嵌入式系统。

- **s390x：** 常用于`IBM`主机，提供高性能和可靠性。

</details>

## 📊 功能 || 特性

<details> 
    <summary>⚙️ 展开描述</summary>

- 系统状态监控
- 搜索所有入站连接和客户端
- 主题：`暗色/浅色`
- 支持多用户和多协议
- 支持协议：`VMESS, VLESS, Trojan, Shadowsocks, Dokodemo-door, Socks, HTTP, WireGuard`
- 支持原生XTLS协议：`RPRX-Direct, Vision, REALITY`
- 流量统计、流量限制与过期时间限制
- 可定制的`Xray`配置模板
- 支持通过`HTTPS`访问面板（自定义域名+SSL证书）
- 支持一键申请`SSL证书`及其自动续订
- 更多高级配置选项见面板
- 修正的`API路由`（用户设置通过`API`创建）
- 支持通过面板按不同参数修改配置
- 支持通过面板导出/导入数据库

</details>
