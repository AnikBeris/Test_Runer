## 📊 支持的架构与设备

<details> 
    <summary>⚙️ 展开</summary>

我们的平台为多种架构和设备提供支持，包括但不限于：

- **amd64:** 标准 PC 和服务器的通用架构。
- **x86 / i386:** 桌面和笔记本电脑常用。
- **armv8 / arm64 / aarch64:** 适用于现代移动与嵌入式设备。
- **armv7 / arm / arm32:** 较老的移动与嵌入式设备架构。
- **armv6 / arm / arm32:** 非常老旧设备架构。
- **armv5 / arm / arm32:** 面向早期嵌入式系统。
- **s390x:** 通常用于 IBM 大型机。

</details>

## 📊 功能特性

<details> 
    <summary>⚙️ 展开</summary>

- 系统状态监控
- 所有连接及用户的搜索功能
- 主题支持：`深色 / 浅色`
- 多用户及多协议支持
- 支持协议：`VMESS, VLESS, Trojan, Shadowsocks, Dokodemo-door, Socks, HTTP, WireGuard`
- 原生支持 XTLS 协议：`RPRX-Direct, Vision, REALITY`
- 流量统计及流量限制功能
- 可自定义的 `Xray` 配置模板
- 支持 HTTPS 面板访问（自定义域名 + SSL 证书）
- 一键获取 `SSL 证书` 并支持自动续期
- 更多高级选项请参阅控制面板
- 提供固定 `API 路径` (通过 `API` 创建用户设置)
- 支持通过面板导入/导出数据库

</details>

<h1 align="center"> 
Cloudflare SSL 证书
</h1>

<div align="center">
  <img src="../media/Tutorial/Article_1/cloudflare.jpg" alt="Cloud flare" width="50%">
</div>

脚本管理支持通过 `Cloudflare` 获取 `SSL 证书`。需要以下资料：

<details> 
    <summary> ⚙️ 展开 </summary>

- 注册在 `Cloudflare` 的邮箱
- Cloudflare Global API 密钥
- 域名需通过 `Cloudflare` 指向当前服务器

## 如何获取 Cloudflare Global API 密钥：

1. 在终端中运行 `x-ui`，选择 `Cloudflare SSL Certificate`。

2. 打开链接：[Cloudflare API Tokens](https://dash.cloudflare.com/profile/api-tokens)

3. 点击 `View Global API Key` (如下图所示)：

<div align="center">
  <img src="../media/Tutorial/Article_1/APIKey1.png" alt="API Key 1" width="70%">
</div>

4. 可能需要重新验证身份，完成后密钥将显示如下图：

<div align="center">
  <img src="../media/Tutorial/Article_1/APIKey2.png" alt="API Key 2" width="70%">
</div>

输入域名、`email`、`API KEY`，示例如下：

<div align="center">
  <img src="../media/Tutorial/Article_1/DetailEnter.png" alt="Detail Enter" width="70%">

<h1 align="center"> 
安装 3X-UI
</h1>

# 1. 快速安装 3X

<details> 
    <summary> ⚙️ 展开 </summary>

```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh
```

</details> 

# 2. 分步安装 3X

<details> 
    <summary> ⚙️ 展开 </summary>

### 📊 安装所需程序包

1. 更新系统：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装所需程序包：

```sh
apt-get install wget curl openssl qrencode systemd -y
```

### 安装 3X-UI 面板

运行以下命令安装：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

脚本可能提示您：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

输入 `y` 自定义端口，或 `n` 使用随机端口。建议使用非默认端口，如 `8181`。

```sh
Username: 3favnjd8
Password: Msdf823Ll
Port: 40608
WebBasePath: vpkPI6ex9ajesDX
Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

安装完成后，即可通过浏览器访问以下路径：

```sh
http://您的服务器IP地址:端口/WebBasePath (如: http://192.168.0.10:40608/vpkPI6ex9ajesDX)
```
