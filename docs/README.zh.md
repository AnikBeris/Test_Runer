## 📊 支持的体系架构和设备

<details> 
    <summary>⚙️ 展开说明</summary>

我们的平台支持多种架构和设备，能够适应各种计算环境。以下是我们主要支持的架构：

- **amd64:** 此架构广泛适用于个人电脑和服务器，能够兼容大多数现代操作系统。

- **x86 / i386:** 常用于台式机和笔记本电脑，支持多种操作系统和应用程序，例如 `Windows, macOS 和 Linux`。

- **armv8 / arm64 / aarch64:** 为现代移动和嵌入式设备设计，如智能手机和平板电脑。设备示例包括：`Raspberry Pi 4, Raspberry Pi 3, Raspberry Pi Zero 2/Zero 2 W`，`Orange Pi 3 LTS` 等。

- **armv7 / arm / arm32:** 用于较老的移动和嵌入式设备，常应用于设备如 `Orange Pi Zero LTS, Orange Pi PC Plus, Raspberry Pi 2` 等。

- **armv6 / arm / arm32:** 面向较早的嵌入式设备，很少见但仍应用于例如：`Raspberry Pi 1, Raspberry Pi Zero/Zero W`。

- **armv5 / arm / arm32:** 更旧的架构，多见于早期嵌入式系统，如早期版本的 `Raspberry Pi` 及一些旧设备。

- **s390x:** 通常用于 `IBM` 主机，为企业计算提供高性能和可靠性。

</details>

## 📊 功能与特性

<details> 
    <summary>⚙️ 展开说明</summary>

- 系统状态监控
- 可搜索所有入站连接和客户端
- 支持主题：`暗色/亮色`
- 支持多用户和多协议

- 支持基于 `HTTPS` 面板访问（自定义域名 + SSL 证书）
- 支持一键申请 `SSL 证书` 并自动续订
- 更多高级配置选项请参阅面板
- 修正的 `API 路由`（通过 `API` 配置用户设置）
- 支持通过面板更改多项参数设置
- 支持通过面板导入/导出数据库

</details>

管理脚本包含通过 `Cloudflare` 申请 SSL 证书的内置功能。要使用脚本获取证书，您需要：


<details>

- 使用 `Cloudflare` 转发的域名（需在 DNS 中指向当前服务器）

<h1 align="center"> 
安装 3X-UI
</h1>


# 1. 快速安装 3X

<details> 
    <summary> ⚙️ 展开说明 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh

```

</details> 

# 2. 分步骤安装 3X


<details> 
    <summary> ⚙️ 展开说明 </summary>

# 📊 安装必要的软件包

1. 系统更新
在安装之前，请确认您的系统已经更新。执行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装必要的软件包

确保您的服务器安装了所需的软件包：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装 3X-UI 面板

在服务器上运行以下脚本以安装面板：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装过程中，脚本会弹出以下提示：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

如果希望手动设置端口则选择 `y`，否则选择 `n`，脚本会随机分配端口。
避免使用标准端口 `22, 80, 8080`。建议选择其他端口，例如 `8181`。


```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

完成这些步骤后，即成功安装面板，可以通过浏览器访问以下 URL 连接到面板：

```sh
http://您的服务器_IP_地址:端口号/WebBasePath（例如： http://192.168.0.10:40608/vpkPI6ex9ajesDX）
```

在浏览器中进入地址后，会跳转到面板的登录页面。请按脚本预先提供的用户登录名和密码进行登录。

## DNS 设置

<details> 
    <summary> ⚙️ 展开说明 </summary>

进入 `X-Ray 设置` -> 选择 `DNS` -> 打开子选项卡 `DNS`：

<div align="center">
  <img src="../media/Tutorial/Article_3/Setting_Xray_DNS.png" alt="Setting Xray DNS" width="70%">
</div>

点击 `创建 DNS`，然后依次添加 `DNS 地址`：

```bash
tcp://8.8.8.8
```

```bash
tcp://1.1.1.1
```

```bash
tcp://9.9.9.9
```

  <img src="../media/Tutorial/Article_3/DNS.png" alt="DNS" width="70%">
</div>
