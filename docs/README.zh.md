## 📊 支持的架构与设备

<details> 
    <summary>⚙️ 展开描述</summary>

我们的平台支持多种计算架构与设备，以确保在不同计算环境中灵活使用。以下是主要支持的架构：

- **amd64:** 这种常见的架构是个人电脑和服务器的标准架构，能够无缝运行现代操作系统。

- **x86 / i386:** 广泛用于台式机和笔记本电脑。操作系统和应用均广泛支持，例如 `Windows, macOS 和 Linux`。

- **armv8 / arm64 / aarch64:** 设计用于现代移动和嵌入式设备，如智能手机和平板电脑。设备实例包括：`Raspberry Pi 4, Raspberry Pi 3, Raspberry Pi Zero 2/Zero 2 W`, `Orange Pi 3 LTS` 等。

- **armv7 / arm / arm32:** 用于较早的移动和嵌入式设备。仍广泛用于 `Orange Pi Zero LTS, Orange Pi PC Plus, Raspberry Pi 2` 等。

- **armv6 / arm / arm32:** 针对非常古老的嵌入式设备。尽管较为罕见，但仍可应用于设备如 `Raspberry Pi 1, Raspberry Pi Zero/Zero W`。

- **armv5 / arm / arm32:** 更旧的架构，主要与早期嵌入式系统相关。如今很少见，可能在旧版 `Raspberry Pi` 或旧智能手机中使用。

- **s390x:** 此架构通常用于 `IBM` 的大型机，大企业工作负载中表现优越。

</details>

<h1 align="center"> 
3X-UI 安装
</h1>


# 1. 快速安装 3X

<details> 
    <summary> ⚙️ 展开描述 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh

```

</details> 

# 2. 分步安装 3X


<details> 
    <summary> ⚙️ 展开描述 </summary>

# 📊 安装所需软件包

1. 系统更新
在安装前，确保您的系统已更新。执行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装必要的软件包

确保您的服务器已安装以下软件包：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装 3X-UI 面板

在服务器上启动安装脚本，执行以下命令：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装过程中，脚本会提示如下问题：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

如果希望手动设置端口请输入 `y`，否则输入 `n`，脚本会自动设置端口。
请勿选择默认端口 `22, 80, 8080`，建议选用其它端口，例如 `8181`。


```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

完成上述步骤后，面板便安装完成。通过浏览器访问指定路径连接到面板：

```sh
http://您的服务器IP地址:端口/WebBasePath（例如: http://192.168.0.10:40608/vpkPI6ex9ajesDX）
```

进入面板地址后，在登录界面输入脚本生成的用户名和密码即可。
