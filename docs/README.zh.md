## 📊 支持的架构与设备

<details> 
    <summary>⚙️ 展开描述</summary>

我们的平台兼容广泛的架构和设备，为各种计算环境提供灵活性。以下列出了主要支持的架构：

- **amd64:** 广泛使用的架构，适用于个人电脑和服务器，大多数现代操作系统都能流畅运行。

- **x86 / i386:** 广泛应用于桌面电脑和笔记本电脑。此架构受众多操作系统和应用支持，包括 `Windows、macOS 和 Linux`。

- **armv8 / arm64 / aarch64:** 专为现代移动和嵌入式设备设计，例如智能手机和平板电脑。设备示例：`Raspberry Pi 4、Raspberry Pi 3、Raspberry Pi Zero 2/Zero 2 W`, `Orange Pi 3 LTS` 等。

- **armv7 / arm / arm32:** 为较旧的移动和嵌入式设备服务。仍广泛用于 `Orange Pi Zero LTS, Orange Pi PC Plus, Raspberry Pi 2` 等设备。

- **armv6 / arm / arm32:** 针对非常老旧的嵌入式设备。虽然较少见，但仍应用于例如 `Raspberry Pi 1, Raspberry Pi Zero/Zero W`。

- **armv5 / arm / arm32:** 较老的架构，主要与早期嵌入式系统相关。今天很少见，但可能在过时的设备中使用，如早期的 `Raspberry Pi` 和部分早期智能手机。

- **s390x:** 此架构通常用于 IBM 的大型主机，为企业工作提供高性能和可靠性。

</details>

## 📊 功能 || 特点

<details> 
    <summary>⚙️ 展开描述</summary>

- 系统状态的监控

- 修复的 `API 路由`（用户设置通过 `API` 创建）
- 支持基于面板修改配置的各种参数
- 支持通过面板导出/导入数据库

</details>

<details>

安装3X-UI
</h1>


# 1. 快速安装3X

<details> 
    <summary> ⚙️ 展开描述 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh

```

</details> 

# 2. 分步骤安装3X


<details> 
    <summary> ⚙️ 展开描述 </summary>

# 📊 安装必要的软件包

1. 系统更新
在开始安装之前，请确保您的系统已更新。执行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装必要的软件包

确保您的服务器已安装必须的软件包：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装控制面板3X-UI

要在服务器上安装控制面板，请执行以下命令：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装过程中，脚本将会询问：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

如果想自定义端口，请选择 `y`，否则选择 `n`，脚本将自动设置端口。
建议不要使用默认的端口 `22, 80, 8080`，可以选择其他任意端口，例如 `8181`。

以下是示例输出信息：

```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

完成以上步骤后，面板安装完成，您可以通过以下路径通过浏览器连接面板：

```sh
http://您的服务器IP地址:端口/WebBasePath(例: http://192.168.0.10:40608/vpkPI6ex9ajesDX)
```

在浏览器中输入地址后，您将进入面板登录页面，输入脚本生成的用户登录名和密码即可。
