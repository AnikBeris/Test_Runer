## 📊 支持的架构和设备

<details> 
    <summary>⚙️ 展开描述</summary>

我们的平台支持广泛的硬件架构和设备范围，能够在多种计算环境中提供灵活性。以下是我们支持的主要架构：

- **amd64:** 这种普遍的架构是个人电脑和服务器的标准，支持大部分现代操作系统。

- **x86 / i386:** 常用在台式机和笔记本电脑中。这种架构得到包括 `Windows, macOS 和 Linux` 在内的许多操作系统和应用程序的广泛支持。

- **armv8 / arm64 / aarch64:** 适用于现代移动与嵌入式设备，例如智能手机和平板电脑。支持的设备如：`Raspberry Pi 4, Raspberry Pi 3, Raspberry Pi Zero 2/Zero 2 W`, `Orange Pi 3 LTS` 等。

- **armv7 / arm / arm32:** 为老旧移动与嵌入式设备的架构，仍然广泛用于如 `Orange Pi Zero LTS, Orange Pi PC Plus, Raspberry Pi 2` 等设备。

- **armv6 / arm / arm32:** 针对非常老旧的嵌入式设备。尽管使用较少，但仍适用于如 `Raspberry Pi 1, Raspberry Pi Zero/Zero W` 等设备。

- **armv5 / arm / arm32:** 较老的架构，主要应用于早期嵌入式系统。如今较为罕见，但在某些过时设备中可能仍在使用，如早期版本的 `Raspberry Pi` 和一些早期智能手机。

- **s390x:** 这种架构通常用于 `IBM` 大型机，为企业工作负载提供高性能和可靠性。

</details>

管理脚本包含通过 `Cloudflare` 申请 `SSL 证书` 的内置功能。通过脚本获取证书，您需要：


<details> 
    <summary> ⚙️ 展开描述 </summary>

- 注册于 `Cloudflare` 的电子邮箱
- 全球 API 密钥（Global API Key）
- 域名需通过 `Cloudflare` 指向当前服务器（配置在 DNS 中）

2. 打开以下链接：[Cloudflare API Tokens](https://dash.cloudflare.com/profile/api-tokens)

3. 点击 `View Global API Key`（见下图）：

安装 3X-UI
</h1>


# 1. 快速安装 3X

<details> 
    <summary> ⚙️ 展开描述 </summary>


```sh
sudo apt update && sudo apt upgrade -y && \
sudo apt install -y git curl openssl qrencode systemd && \
rm -rf self_signed_certificate.sh && \
curl -O https://raw.githubusercontent.com/AnikBeris/self-signed-certificate/main/self_signed_certificate.sh && \
chmod +x self_signed_certificate.sh && \
bash ./self_signed_certificate.sh

```

</details> 

# 2. 按步骤安装 3X


<details> 
    <summary> ⚙️ 展开描述 </summary>

# 📊 安装必要的软件包

1. 更新系统
在开始安装之前，请确保您的系统是最新的。运行以下命令：

```sh
sudo apt update && sudo apt upgrade -y &&
```

2. 安装必要的软件包

确保服务器上安装了所需的软件包：

```sh
apt-get install wget curl openssl qrencode systemd -y
```



## 安装 3X-UI 面板

在服务器上运行以下脚本执行安装命令：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

安装过程中，脚本会问：

```sh
Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
```

选择 `y` 自定义端口，或者 `n` 让脚本自动设置端口。
不要使用标准端口 `22, 80, 8080`，建议选择其他端口，如 `8181`


```sh
Username: 3favnjd8

Password: Msdf823Ll

Port: 40608

WebBasePath: vpkPI6ex9ajesDX

Access URL: http://192.168.0.149:40608/vpkPI6ex9ajesDX
```

安装完成后，您可以通过以下路径使用浏览器连接到面板：

```sh
http://您的服务器IP地址:端口/WebBasePath(示例: http://192.168.0.10:40608/vpkPI6ex9ajesDX)
```

在浏览器中输入该地址后，将出现登录窗口，您需输入脚本运行时提供的用户名和密码。
